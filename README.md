# DDL_Reminder

## 使用方法

1、fork本项目

2、配置github secret

3、开启github action推送

## 参数配置

> 必须配置的内容

```bash
ZJU_USERNAME : 'username',
ZJU_PASSWORD : 'password',
```

> 可选配置内容

<font size="3">钉钉机器人的推送</font>

```bash
'DD_BOT_SECRET': '',      # 钉钉机器人的 DD_BOT_SECRET，加签密钥
'DD_BOT_TOKEN' : '',      # 钉钉机器人的 DD_BOT_TOKEN, access_token后一段
```
